# Visual Baselines

This directory stores Terminal Cockpit review evidence for TC-001 through TC-008.

- `current-split-desktop.png` - current split/terminal view before TC-002 shell edits.
- `current-map-desktop.png` - current canvas/map view before TC-005 map edits.
- `reference-cockpit.png` - target reference screenshot for proportion and hierarchy comparison.
- `comparison-notes.md` - notes comparing current screenshots, the reference, and the design contract.
- `tc-002-left-files-right-ops.png` - shell pass before the one-sidebar correction.
- `tc-002-shell-desktop.png` - centered command cockpit shell pass.
- `tc-003-one-sidebar-linked-terminal.png` - one-sidebar navigation and linked terminal pass.
- `tc-004-terminal-surface.png` - tactical terminal surface pass.
- `tc-005-strategic-map.png` - strategic operations map pass.
- `tc-006-command-menu.png` - command-first workspace action menu pass.

If the reference image cannot be committed, keep `reference-cockpit.png` out of
the repo and describe the source plus observed traits in `comparison-notes.md`.
